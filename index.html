Follow me on this journey as I try to learn out to build websites. I originally started a blog some years bach when was intrigued by how easy it is to make a blog. However it did not last as I was too lazy to keep updating the website. The website was eventually taken over by spammers and I lost the domain name This is different, I am currently pursuing a degree in Software Engineering at harvard extension school and learning how to build a website seems like a natural progression. Unlike my first merry go round, I am actually building this website myself and in the process I expect to learn HTML and CSS. The programs I am proficient in are java and python but I also know a little bit of C. My long term goal is not neccesarilly to be proficient in one program or operating system, I want to be a flexible programmer that can do a bunch of things and turn my abilities into something tangible.
